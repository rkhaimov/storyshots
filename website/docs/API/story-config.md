# StoryConfig

Настройки для текущей истории.

---

## device

Указывает на текущее устройство.

## journal

Ссылается на закреплённый экземпляр [Journal](#).

## testing

Указывает режим, в котором запущена история.

- `true`: История выполняется как тест.
- `false`: История запущена в режиме [предпросмотра](#).

:::note
Данное свойство полезно для контроля [внешней среды](#). Например, для отключения анимаций и других недетерминированных
действий/данных, чтобы обеспечить стабильные и надёжные результаты тестов.
:::
