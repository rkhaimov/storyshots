Являясь частью секции [*аргументов*](/specification/requirements/borders#определение-границ) данный блок описывает
последовательность действий, которые пользователь предпринимает на странице.

## Связь с библиотекой

[//]: # (TODO: Добавить ссылки)

В библиотеке `storyshots` объекты `actor` и `finder` совместно представляют пользователя — агента, способного выполнять
различные действия на странице.

:::note
С точки зрения спецификации, действия пользователя можно представить в виде *значений* (например, тех, которые можно
записать в переменную или передать в функцию). `storyshots` использует этот подход, приближая общую архитектуру к
функциональной парадигме, что положительно сказывается на расширяемости решения в целом.
:::
