# Монитор

<BalancedMetricsTip improves={[Metric.RegressionProtection, Metric.RefactoringAllowance, Metric.Maintainability]} />

Монитор включает основной результат работы AUT — изображение интерфейса.
Относится к секции [*результата*](/specification/requirements/borders#определение-границ).

## Верификация

Схема процесса выглядит следующим образом:

<Diagram src={require('./assets/screenshot.drawio.png')} />

:::note
Программа не генерирует изображение напрямую. Вместо этого она взаимодействует с DOM-деревом, и браузер отрисовывает
итоговый экран для пользователя.
:::

## Связь с библиотекой

`storyshots` реализует [актора](/API/story-elements/actor), который комбинирует автоматическое создание изображений через снимки экрана с
возможностью создания промежуточных снимков.
