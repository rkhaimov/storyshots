# Спецификация

Фундамент архитектуры решения и техники организации процесса тестирования строится на базовых требованиях. Определить их
можно, начав с анализа [основной роли тестов](/specification/main-role) в разработке ПО, а также проблем, возникающих
при верификации пользовательских интерфейсов.

:::note
**Требование** в данном контексте — это утверждение, которое обязательно должно соблюдаться для выполнения описываемой
спецификации.
:::
